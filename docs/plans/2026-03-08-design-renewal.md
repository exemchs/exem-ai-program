# 디자인 리뉴얼 설계

## 방향
구조 유지 + 디자인 톤 업그레이드 + 카피 압축

## 톤
MUJI(담백, 절제, 사실) + 현대카드(리듬감, 부드러운 단정, 여유)
→ 담담하게 말하는데, 한 문장이 귀에 걸리는 톤

## 1. Hero — 그래디언트 스크롤

- Unicorn Studio 제거
- 240vh 높이 컨테이너, 타이틀 sticky 고정
- 오렌지 계열 그래디언트: #1a0a00 → #2d1810 → #8b4513 → #e67e22 → #f4c57a → #fef9f0
- 스크롤에 따라 그래디언트가 서서히 밝아짐
- 배지: "Exem Claude Code Class" 유지
- 타이틀: "'나중에 해야지' 했던 Claude Code, 지금이 기회입니다." 유지
- 서브카피: "Claude Code, 같이 시작하면 됩니다."

## 2. 카피 압축

### WhyNow
- "매일 기술을 만드는 사람들" → "기술을 만드는 사람이, 도구를 가장 잘 압니다."
- "앱 채팅을 넘어서는 시점" → "채팅은 해봤습니다. 그 다음이 궁금한 시점입니다."
- "미루면 격차가 됩니다" → "도구는 미루는 순간, 거리가 됩니다."

### WhatWeLearn
- 메인: "코드가 아니라, 일하는 방식을 바꾸는 겁니다."
- 서브: "하고 싶은 일을 말로 전하면 됩니다."

### OneMoreThing
- "하나를 익히면, 그 다음은 스스로 찾게 됩니다."

### Footer
- 메인: "엑셈의 내일은, 오늘 시작됩니다." (유지)
- 서브: "처음이라 같이 합니다. 그래서 쉬워집니다."

## 3. 디자인 톤 업그레이드

- 타이포: tracking 더 타이트, Hero 폰트 사이즈 업
- 여백: 섹션 간 py-32 → py-40, 카드 내부 여백 증가
- 카드: border 투명도 낮추기, hover glow 더 은은하게
- 전반적으로 넓고 여유 있는 레이아웃
